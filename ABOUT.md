## Nama Aplikasi
StockWise - Inventory Management System

## Deskripsi
StockWise adalah aplikasi manajemen inventaris yang dibangun menggunakan JavaFX dan MySQL. Aplikasi ini dirancang untuk membantu bisnis dalam mengelola stok produk, transaksi masuk dan keluar, serta manajemen pengguna dengan mudah dan efisien.

## Fitur Utama
- **Manajemen Pengguna**: Sistem login dan registrasi dengan peran ADMIN dan STAFF
- **Dashboard**: Tampilan utama untuk melihat ringkasan inventaris
- **Manajemen Produk**: Tambah, edit, hapus, dan lihat produk dengan kategori, harga, dan stok
- **Manajemen Transaksi**: Catat transaksi masuk (IN) dan keluar (OUT) produk
- **Import/Export CSV**: Impor dan ekspor data produk dalam format CSV
- **Keamanan**: Enkripsi password menggunakan BCrypt
- **Antarmuka GUI**: Interface yang user-friendly menggunakan JavaFX

## Teknologi yang Digunakan
- **Bahasa Pemrograman**: Java 21
- **Framework UI**: JavaFX 21
- **Database**: MySQL
- **Build Tool**: Maven
- **Library Tambahan**: jBCrypt untuk hashing password
